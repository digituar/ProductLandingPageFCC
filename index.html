WHY DIGITUAR

Previously, many files had to be used on the stage. Aside from carrying so many files, it was not easy at all to find a desire we were looking for. Since we experienced these problems on the stage ourselves, we analyzed all the necessary conditions for a suitable, practical software for the scene! From time to time, every musician, repertoire on stage, concert, folk bar, wedding, etc. And it had gone through. For these reasons, the Repertoire Program has been formed. The repertoire program was designed according to the most professional musician from the most amateur musician, and was developed as a software that can be used easily in concert, on stage, wherever the repertoire is required. At the same time, the most necessary practical applications have been implemented by using different musicians on stage.

Genres

You can save all the works under a certain category and then search under that category.

Import/export

All of the works can be exported individually or can be imported again on another computer.

3 different languages

It supports 3 different languages ​​(English, German, Turkish). You can also set the languages ​​yourself and change the texts.

Multimedia

You can add not only the words of the work, but also the note (BMP, GIF, TIF, JPG, PDF) and Media file (Mp3, Wave, MIDI).
